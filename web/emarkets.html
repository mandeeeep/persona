eMarkets

eMarkets is a web portal which provides information about the various electronic devices namely mobile and handheld devices, chiefly mobile phones. Its main purpose is to let the user know about the upcoming and existing mobile phones from different brands more like a wiki, apart from these functionalities it allows the user to register to the site and post within the info about their own devices which they would prefer to sell or market. Thus, making eMarkets a kind of marketing-hub.
